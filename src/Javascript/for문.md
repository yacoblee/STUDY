# for문

> #### for문을 사용하는 이유는? 불필요한 단순 반복행위를 줄이기 위해서 사용한다. for문을 사용하면 긴 코드를 짧게 줄일 수 있다!


<br>



## for문 구성

```js

for(초기문; 조건문; 증감문;) {
  반복조건이 맞으면 실행되는 코드 
}

```

#### 예시)


```js

for (let i=0; i < 5; i++){
  console.log(i);
}

```

위와 같이 조건문을 실행했을 때 0부터 4까지 0, 1, 2, 3, 4 총 5번이 실행된다
